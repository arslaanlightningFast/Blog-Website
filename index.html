About

A small, professional blogging UI — built with only HTML, CSS, and vanilla JavaScript.

What it includes

  • Multi-page feel via hash routing
  • Create, read, update, delete posts
  • Search by title + filter by author
  • Likes and comments stored locally
  • Pagination
  • Dark mode default + toggle saved

How it works

Posts are stored in localStorage, so the data stays on your device. No backend is required. Open index.html in any browser.

Tip: If you want this to become a real product later, we can add a backend API and authentication without changing the UI structure.
